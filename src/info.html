Привет, это инструкция к калькулятору!

Сборка:

● make all - запуск калькулятора и открытие исполняемого объекта
● make install - установка приложения калькулятор
● make unistall - удаление приложения
● make clean - очистка всех ненужных файлов
● make dist - создание архива
● make test - проверка функционала калькулятора с помощью тестов
● make gcov_report - отображение покрытия кода unit тестами
● make dvi - вывод паспорта проекта

Функционал калькулятора:

● арифметика:

сложение (+)
вычитание (-)
умножение (*)
деление (/)

● функции:

остаток от деления (mod)
корень (sqrt)
возведение в степень (^)
логарифм (log)
натуральный логарифм (ln)
синус (sin)
косинус (cos)
тангенс (tan)
арксинус(функция обратная синуса) (asin)
арккосинус(функция обратная косинуса) (acos)
арктангенс(функция обратная тангенса) (atan)

● вычисление выражения с X и построение графика заданной функции

Кроме того, присутствуют кредитный и депозитный калькуляторы
